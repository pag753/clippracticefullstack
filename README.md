

# Credit Card React & Node.JS Form 

## Steps to run me
### Clone me via GitHub:
```
$ git clone https://github.com/pag753/clippracticefullstack
```

### Install me:
```
$ npm i
```

### Run me:
```
$ npm run start
```

### Show me locally
Open [http://localhost:3000](http://localhost:3000) to see me in the browser

## Test me:
```
$ npm run test
```
